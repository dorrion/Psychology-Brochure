## Psychology Brochure
안녕하세요. 아주대학교 심리학과 요람 프로젝트를 진행하고 있는 구르는 감자단입니다!

요람 프로젝트의 목적은 다음과 같습니다.

- 아주대학교 심리학과 학우들의 진로 선택에 도움
- 아주대학교 심리학과 생활에 도움이 될 수 있는 정보 제공

배포 주소 : https://psychology-brochure.vercel.app/

## 프로젝트 기간
